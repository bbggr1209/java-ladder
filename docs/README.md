## 구현할 기능 목록

- [ ] 플레이어 이름 입력
  - [ ] 이름을 입력 받을 메세지 출력
  - [ ] 이름을 입력 받음
  - [ ] 이름을 쉼표(,)를 기준으로 분리
  - [ ] 이름의 유효성 검사
    - [ ] 이름이 5글자 이하여야 한다.
    - [ ] 이름에는 공백이 들어갈 수 없다.
    - [ ] 이름이 중복되어서는 안된다.

- [ ] 사다리 높이 입력
  - [ ] 사다리 높이 입력 메세지 출력
  - [ ] 사다리 높이 입력 받음
  - [ ] 사다리 높이의 유효성 검사
    - [ ] 사다리 높이는 1 이상이어야 한다.

- [ ] 결과 출력
  - [ ] 플레이어 이름 출력 
  - [ ] 사다리 출력

--- 

## 필요한 객체

- Player
  - 이름을 입력받고 유효성 검사를 수행하는 객체

- Players
  - Player 객체를 관리하는 객체

- Line
  - 사다리의 선을 관리하는 객체

- Ladder
  - 여러 개의 선 객체를 관리하는 객체

- 입출력
  - 입력을 받고 출력하는 객체
